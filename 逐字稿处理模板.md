# 逐字稿处理模板（简化版）

## 使用说明

这个模板用于处理课程逐字稿，自动去掉语气词，保留重点内容，按结构整理。

**特点：**
- ✅ 不需要选择时间段
- ✅ 直接复制逐字稿内容
- ✅ 自动矫正术语错误
- ✅ 自动提取主题并命名文件
- ✅ 自动整理成 Markdown 格式

## 步骤

### 1. 粘贴逐字稿内容
直接粘贴你要处理的逐字稿内容：

```text
（粘贴逐字稿内容）
```

### 2. 文件名（自动提取）
AI 会根据逐字稿内容自动提取主题并生成文件名，例如：
- `松月-Obsidian介绍-20260525.md`
- `Part3-知识库搭建-方法论.md`
- `一人公司商业模式-课程笔记.md`

（你也可以在指令中指定文件名）

### 3. 执行处理指令
复制以下指令，发送给 AI：

```
请帮我处理这段逐字稿：
1. 去掉语气词（嗯、啊、那个、哈哈、哎呀等）
2. **保留所有内容** - 不要删减任何核心信息，保留讲师的完整意思和所有细节
3. 按照原有的结构整理（章节、要点等）
4. **不要精简** - 保留讲师说的每一句话、每一个例子、每一个细节
5. 整理成清晰的 Markdown 格式
6. **矫正术语错误** - 自动识别并矫正逐字稿中的各种术语错误，例如：
   - Cloud Code / cloud code → Claude Code
   - opc → Obsidian（根据上下文判断）
   - 艺人公司 → 一人公司
   - 其他术语错误也会自动识别和矫正
7. 根据内容自动提取主题，生成文件名（格式：主题-日期.md）

逐字稿内容：
（粘贴逐字稿内容）
```

（AI 会自动识别并矫正各种术语错误，生成文件名）

### 4. 保存结果
将处理后的内容保存到 AI 训练营文件夹，使用你自定义的文件名。

## 文件命名规范

### 自动命名格式
AI 会根据逐字稿内容自动提取主题并生成文件名：

**格式：** `主题-日期.md`

**示例：**
- `松月-Obsidian介绍-20260525.md`
- `Obsidian知识库搭建-20260525.md`
- `一人公司商业模式-课程笔记-20260525.md`

### 命名规则
1. **提取主题** - 从逐字稿中识别核心主题
2. **添加讲师** - 如果有讲师信息，包含讲师名
3. **添加日期** - 自动添加当前日期
4. **添加类型** - 可选：课程笔记、方法论等

### 手动指定文件名
如果 AI 自动生成的文件名不满意，可以在指令中指定：
```
请使用以下文件名保存：
[指定文件名].md
```

## 术语矫正规则

AI 会自动识别并矫正逐字稿中的术语错误，包括但不限于：

### 常见术语矫正示例
| 错误术语 | 正确术语 | 说明 |
|---------|---------|------|
| Cloud Code / cloud code | Claude Code | AI 工具名称 |
| opc | Obsidian | 笔记工具（根据上下文判断） |
| skill / skills | Skills | AI 工具功能 |
| 艺人公司 | 一人公司 | 商业模式术语 |
| ... | ... | 其他术语错误 |

**重要说明**：
- AI 会根据上下文自动识别术语错误并进行矫正
- 不仅限于以上示例，会识别各种术语错误
- 如果你有特定的术语需要矫正，可以在指令中特别说明

## 注意事项

1. **不要精简内容** - 保留讲师说的每一句话、每一个例子、每一个细节
2. **去掉语气词** - 嗯、啊、那个、哈哈、哎呀等（只去掉语气词，不删减内容）
3. **按结构整理** - 保持原有的章节结构
4. **矫正术语错误** - AI 会自动识别并矫正各种术语错误，不仅限于示例中的几个
5. **保留完整意思** - 不要删减任何核心信息，保留讲师的完整意思
6. **保存到 AI 训练营** - 统一存放处理后的逐字稿
7. **自动提取主题** - AI 会根据内容自动命名文件
